# SmartTrack AI

SmartTrack AI is a modern attendance tracking system demo that combines a sleek web UI with an AI-assisted backend for smarter classroom and lab management.

## Features

- Clean, responsive dashboard-style UI for attendance.
- Quick student search and filtering.
- Status chips for Present, Late, and Absent.
- Daily summary card with total students and present count.
- Designed to plug into a Node.js + SQLite backend later.

## Tech stack

- Frontend: HTML5, CSS3 (glassmorphism-style dashboard layout).
- Backend (planned): Node.js with Express.
- Database (planned): SQLite for lightweight storage.

## Project goals

- Make marking attendance fast and visually clear for teachers.
- Provide a simple codebase you can extend into a full-stack project.
- Experiment with AI-assisted suggestions (e.g., highlight frequently absent students) in future iterations.

## Getting started (UI only)

1. Clone or download this repository.
2. Open `index.html` in a browser.
3. Explore the static dashboard and layout.

Backend setup and AI features will be added in upcoming versions.
